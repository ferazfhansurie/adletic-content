# 7-Day Viral Content Plan — Studio Model + Voice Overlay Format

**Format:** Studio-lit model video with a visual "wait, what?" element + voice overlay narration
**Platforms:** Instagram Reels / TikTok
**Duration:** 15-30 seconds each
**Style:** Clean, cinematic, slow-motion reveals. Think editorial fashion meets internet brain rot.

---

**The Hook (first 1.5s):** Model mid-stride, oversized suit visible immediately.
