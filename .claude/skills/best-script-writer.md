# Skill: Best Script Writer

Write scripts that stop thumbs and hold attention. Built for short-form AI video content (15-60s). Not generic copywriting — this is performance writing for the scroll.

## Philosophy
- Every line earns the next second of watch time
- Write for the EAR first, screen second
- Malaysian English cadence — direct, punchy, code-switches naturally
- The script IS the video. If the words don't carry without visuals, rewrite.

## Proven Script Structures

### 1. HOOK → TENSION → PAYOFF → CTA (15-30s)
Best for: Bold claims, reveals, controversial takes
```
HOOK (0-2s): Pattern interrupt. Stop the scroll.
TENSION (2-15s): Build the open loop. Stack curiosity.
PAYOFF (15-25s): Deliver the insight. Close the loop.
CTA (25-30s): One clear action. No fluff.
```

### 2. PROBLEM → AGITATE → SOLVE → CTA (30-60s)
Best for: Tutorials, "how I did it", educational
```
PROBLEM (0-5s): Name their exact pain. Be specific.
AGITATE (5-20s): Twist the knife. Show the cost of not solving it.
SOLVE (20-50s): Step-by-step or reveal. Make it feel achievable.
CTA (50-60s): Tell them what to do RIGHT NOW.
```

### 3. STORY → PIVOT → LESSON (30-60s)
Best for: Personal brand, relatable content, community building
```
STORY (0-20s): Start mid-action. No "So basically..."
PIVOT (20-40s): The moment everything changed.
LESSON (40-55s): What they should take away.
CTA (55-60s): Connect to their life.
```

### 4. CONTRAST STACK (15-30s)
Best for: Before/after, AI reveals, comparison content
```
BEFORE (0-10s): Show the old way. Make it feel painful.
AFTER (10-20s): Show the new reality. Make it feel effortless.
BRIDGE (20-30s): "Here's how" — connect to actionable next step.
```

## Hook Formulas That Work

### Curiosity Hooks
- "Nobody's talking about this but..."
- "I found out why [surprising thing] and it changes everything"
- "This is the [thing] that [unlikely result]"

### Confrontation Hooks
- "Stop [common behavior]. Here's why."
- "[Popular advice] is wrong. Here's what actually works."
- "If you're still [doing X], you're already behind."

### Identity Hooks
- "POV: You're the [identity] who [does thing]"
- "This is for the [specific person] who [specific situation]"
- "Only [X%] of people know this"

### Proof Hooks
- "I [achieved result] in [timeframe]. Here's the breakdown."
- "This took me from [before state] to [after state]"
- "[Number] that changed my [career/life/workflow]"

## Writing Rules

### Rhythm & Pacing
- **2-8 words per line.** If a line has more than 10 words, break it.
- **Vary sentence length.** Long. Short. Medium length for explanation. Short.
- **Use line breaks as beats.** Each line break = a breath = a visual cut.
- **Front-load power words.** "Money." not "The financial aspect of..."
- **Read it out loud.** If you stumble, rewrite. If it feels slow, cut.

### Word Choice
- Use "you" more than "I"
- Active voice only. Never passive.
- Concrete > abstract. "RM5,000" > "significant income"
- Cut adverbs. "Quickly ran" → "Sprinted"
- No filler: "basically", "actually", "like", "so", "um", "honestly"
- Malaysian-natural: "lah", "kan", code-switch if it fits the creator persona

### CTA Rules
- One CTA only. Never two.
- Make it specific: "Save this" / "Follow for part 2" / "Comment PROMPT"
- Never "Like and subscribe" — that's dead.
- Best CTAs create FOMO or community: "Join us before we close"

## Output Format
```
---
TITLE: [Working title]
DURATION: [15s / 30s / 60s]
STRUCTURE: [Which framework]
HOOK TYPE: [Curiosity / Confrontation / Identity / Proof]
MUSIC: [Mood suggestion]
---

HOOK (0-Xs):
[Voiceover line]
[Visual direction]

BODY (X-Xs):
[Line by line with visual notes]

CTA (X-Xs):
[Final line + visual]
---
```

## Save to
`outputs/scripts/` with format: `{topic}-{duration}s-{date}.md`

## Anti-Patterns (Never Do These)
- Starting with "Hey guys" or "What's up"
- Telling people what you're about to tell them
- Using "journey" unironically
- Ending with "Let me know in the comments what you think"
- Writing more than 3 sentences before the first value drop
- Using words your audience wouldn't use in real life
