# Skill: Content Calendar

Plan a week or month of content for social media (Instagram, TikTok, YouTube).

## Input
Ask the user for:
- **Period** — This week / next 2 weeks / this month
- **Platform(s)** — Instagram, TikTok, YouTube, all
- **Goal** — Grow followers, drive waitlist signups, nurture community, launch course
- **Content pillars** — e.g., tutorials, behind-the-scenes, results, tips, memes

## Process
1. Map out posting schedule (frequency per platform)
2. Assign content type to each day (educational, entertaining, promotional, engagement)
3. Write a one-line concept for each post
4. Flag which posts need video, image, carousel, or text
5. Mark promotional posts (max 20% of total — don't oversell)

## Output
Save to `outputs/calendars/` as a markdown table.
