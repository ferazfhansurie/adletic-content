# Skill: Video Script Generator

Generate scroll-stopping short-form video scripts for Reels, TikTok, or ads.

## Input
Ask the user for:
- **Topic or product** — What is the video about?
- **Format** — Hook-story-CTA / problem-solution / tutorial / testimonial
- **Duration** — 15s / 30s / 60s
- **Platform** — Instagram Reels, TikTok, YouTube Shorts, or ad
- **Tone** — Hype, educational, raw/authentic, cinematic

## Process
1. Write 3 hook options (first 1-3 seconds — the make-or-break moment)
2. Write the full script with visual direction and on-screen text cues
3. Include a strong CTA at the end
4. Keep it tight — short sentences, punchy lines, no filler

## Output
Save to `outputs/scripts/` with descriptive filename.

## Format
```
HOOK (0-3s): [What they see + what they hear]
BODY (3-Xs): [Scene-by-scene breakdown with visuals + voiceover/text]
CTA (last 3s): [Clear call to action]

ON-SCREEN TEXT: [Key text overlays]
MUSIC/SFX: [Suggested mood or sound]
```
